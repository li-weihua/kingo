🍎🍌 苹果香蕉价格计算 🍌🍎

题目：

🍎🍎 + 🍌 = 10元
🍎 + 🍌 = 6元

求：🍎和🍌的价格

🤔 我们来想想：

第一步：观察两个算式

🍎🍎 + 🍌 = 10元
🍎 + 🍌 = 6元

💡 发现：第一个算式比第二个算式多了一个 🍎

第二步：找出一个苹果的价格

第一个算式：🍎🍎 + 🍌 = 10元

第二个算式：🍎 + 🍌 = 6元

所以：多出来的 🍎 = 10元 - 6元 = 4元

第三步：找出一个香蕉的价格

我们知道：🍎 = 4元

从第二个算式：🍎 + 🍌 = 6元

所以：🍌 = 6元 - 4元 = 2元

第四步：检查答案对不对

检查第一个算式：🍎🍎 + 🍌 = 4元 + 4元 + 2元 = 10元 ✓

检查第二个算式：🍎 + 🍌 = 4元 + 2元 = 6元 ✓

🎯 答案：

🍎 = 4元
🍌 = 2元

🎉 太棒了！我们成功解决了这个问题！

💡 记住：先找出多出来的东西，再算出其他的

🍎🍎🍎🍎 + 🍌🍌 = 20元

（4个苹果 + 2个香蕉 = 4+4+4+4 + 2+2 = 16 + 4 = 20元）
